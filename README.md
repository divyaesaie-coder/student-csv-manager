# Student CSV Data Manager

This is a Python project that reads and writes student data using CSV files and calculates total and average marks for each student.

## Technologies Used
- Python
- CSV Module

## Features
- Store student records in CSV format
- Read data from CSV files
- Calculate total marks
- Calculate average marks
- Display student performance data

## Concepts Used
- File Handling
- CSV Module
- Dictionaries
- Loops
- Conditional Statements
- Arithmetic Operations

## Project Type
Python File Handling Project
